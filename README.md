# BookMyStay

BookMyStay is a full-stack accommodation booking platform inspired by Airbnb, developed using the MERN stack. The application 
enables users to browse properties, view detailed listings, and manage accommodation data through a scalable 
client-server architecture.

## Overview

The project demonstrates the implementation of modern web development concepts, including RESTful APIs,
database integration, user authentication, and responsive user interface design.

## Key Features

- User authentication and authorization
- Property listing management
- Detailed property pages
- Image upload support
- Search and filtering functionality
- Responsive user interface
- RESTful API architecture
- MongoDB database integration

## Installation

Clone the repository:

```bash
git clone https://github.com/qanitairam1-rgb/airbnb-clone-bookmystay.git
```

Install dependencies:

```bash
npm install
```

Start the backend server:

```bash
cd api
npm install
npm start
```

Start the frontend application:

```bash
cd client
npm install
npm start
```
